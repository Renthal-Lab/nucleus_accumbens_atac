# Epigenomic profiling of mouse nucleus accumbens at single-cell resolution
This repository contains companion code for Bhatia &amp; Yang et al. (2023). Specifically, we provide code for gene regulatory network (GRN) discovery, and analyses of single-nucleus chromatin accessibility and gene expression data for the mouse nucleus accumbens.
